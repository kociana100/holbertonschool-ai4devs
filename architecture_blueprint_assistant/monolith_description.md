# Monolithic Architecture

- **Frontend App**: Web interface where users log in, add expenses and income, view dashboards, and manage budgets.  
- **Backend Monolith**: Central application containing all business logic, APIs, and internal modules in a single deployable system.  
- **Authentication Module**: Manages user registration, login, sessions, and secure access control.  
- **Expense Management Module**: Handles adding, editing, deleting, and categorizing expense and income records.  
- **Budget Analytics Module**: Calculates spending trends, budget usage, savings summaries, and financial health metrics.  
- **AI Insights Module**: Generates smart spending insights and recommendations based on user financial data.  
- **Subscription Detection Module**: Identifies recurring transactions and tracks subscription-related expenses.  
- **Notification Module**: Sends alerts for budget limits, unusual spending, and recurring payment reminders.  
- **Reporting Module**: Produces summaries, charts, and exportable reports for user financial analysis.  
- **Database**: Stores user accounts, transactions, budgets, categories, subscriptions, and generated insights centrally.
